   Or via dashboard: New ‚Üí Database ‚Üí Add Redis

   Required variables:
   - SECRET_KEY (generate with: openssl rand -hex 32)
   - ENABLE_GPU=false
   - STORAGE_TYPE=local
   - WORKER_CONCURRENCY=1

   Command: railway variables set KEY="value"

   Command: railway up
   Or auto-deploys from GitHub

   Get URL: railway open
